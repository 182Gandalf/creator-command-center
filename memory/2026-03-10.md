## Lessons Learned Today
- `onclick="func('${text}')"` is ALWAYS risky for AI-generated text — always use index reference + stored array
- Mobile click issues: try touch-action:manipulation FIRST before complex debugging
- When mobile button is "unclickable" with no :active response = something physically covers it OR onclick attr has issues
- addEventListener('touchend', e.preventDefault()) + addEventListener('click') at script END is most reliable mobile button binding
- Always move loading state BEFORE any async work — user needs immediate visual feedback on tap
- onboarding.py update path should ALWAYS reset AI learning fields (liked_topics, disliked_topics, tone_preferences)
- `mix-blend-mode: screen` only works for truly transparent backgrounds (black = transparent)
- MP4 with `loop autoplay muted playsinline` loops fine without being a GIF
