FlowCast is an AI content scheduler for creators. Our TikTok integration lets users compose, schedule, and publish videos directly to their profiles.

Products: Content Posting API (Direct Post) publishes videos/photos with captions, hashtags, privacy settings. Login Kit handles OAuth authentication.

Scopes: video.publish publishes content (user-initiated). video.list checks post status. user.info.basic identifies connected accounts.

Flow: Users connect via OAuth, upload content, add captions with AI hashtags, set privacy (public/friends/private), then publish or schedule.

Data: OAuth tokens encrypted with AES-256. We store post metadata only. Users can disconnect anytime.

Safety: For creators managing own accounts only. No spam or bulk posting. Compliant with TikTok Terms.
